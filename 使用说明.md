# 🎮 SS14 汉化工具箱 v3.0 使用指南

> 专为零基础用户设计，只需点击几下鼠标即可完成汉化工作！

---

## 📦 第一步：安装

### 1. 安装 Python
如果电脑没有 Python，请访问 https://www.python.org/downloads/ 下载安装。
> **提示**：安装时务必勾选 "Add Python to PATH"！

### 2. 安装依赖
双击运行 **`0_安装依赖.bat`**，等待窗口显示"依赖安装完成"后关闭。

### 3. 放置位置
将整个工具文件夹的内容复制到你的 **游戏仓库根目录** 下。
例如：`F:\Github\Monolith-CN\ss14_tracker.py`

---

## 🚀 第二步：使用（推荐方式）

### 启动工具
双击 **`Start_GUI.bat`** 打开图形界面。

### 一键同步（最简单！）
1. 在「Paratranz 同步」选项卡填写你的 **项目 ID** 和 **API Token**
2. 点击顶部的 **「⚡ 开始一键同步」** 按钮
3. 等待完成！工具会自动：提取 → 上传 → 下载 → 合并

> **Token 获取方式**：登录 [Paratranz](https://paratranz.cn) → 右上角头像 → 设置 → 个人接口凭证

---

## 🔧 分步操作（可选）

如果你只想执行其中某一步：

| 选项卡 | 功能 | 说明 |
|--------|------|------|
| **1. 提取原文** | 扫描游戏文件 | 生成 `en.json` 供翻译 |
| **2. Paratranz 同步** | 上传/下载 | 与翻译平台同步 |
| **3. 合并翻译** | 写回游戏 | 将 `zh.json` 合并到游戏文件 |
| **⚙️ 设置** | 自定义 | 修改要提取的字段列表 |

---

## ❓ 常见问题

**Q: 提示"找不到目录"？**
A: 请确保工具放在游戏仓库根目录，且该目录下有 `Resources` 文件夹。

**Q: 上传/下载失败？**
A: 检查 Token 是否正确，可点击「测试连接」按钮验证。

**Q: 合并后游戏文件格式乱了？**
A: 不会！工具使用专业库保留所有注释和格式。

---

## 🤖 自动化（GitHub Actions）

仓库已配置自动工作流，每天会：
1. 检查上游更新
2. 自动提取并上传新文本
3. 下载最新翻译并提交

只需在 GitHub 仓库设置 `PARATRANZ_TOKEN` Secret 即可启用。
